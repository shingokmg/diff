# diff

diff は、修正前後の日本語テキスト差分と文字数を PC ブラウザで確認できる静的ツールです。

公開 URL:

- `https://diff.shingokmg.com`

## できること

- BEFORE / AFTER テキストをブラウザ上で見比べる
- 行単位と文字単位を組み合わせて差分を表示する
- AFTER テキストをその場で手直しする
- 差分だけ表示して変更箇所に集中する
- BEFORE の固定と AFTER の全文コピーを使う
- 総文字数を確認する

## 向いている使い方

- AI に修正してもらった文面の確認
- 人にレビューしてもらった文面の確認
- 字数制限のある原稿の文字数確認
- 業務 PC で専用ツールをインストールして使えない環境での利用

## 基本方針

- 入力テキストはブラウザ内で比較します
- 比較処理のためにサーバーへ送信しません
- 入力テキストを localStorage、sessionStorage、Cookie に自動保存しません
- 公開コードで実装を確認できます

詳しくは以下を参照してください。

- [`security.html`](./security.html)
- [`privacy.html`](./privacy.html)

## ローカルで確認する

静的サイトなので、`index.html` をそのままブラウザで開いて確認できます。

ローカルサーバーを使う場合:

```sh
python3 -m http.server
```

その後、`http://localhost:8000` を開いてください。
