Privacy

diff は、入力テキストや差分結果を比較処理のためにサーバーへ送信しないことを前提に設計されています。ここでは、何を送らないか、何を保存しないか、アクセス統計として何が扱われ得るかを整理します。

このページで伝えたいこと

  • 入力テキストはブラウザ内で比較し、比較処理のためにサーバーへ送信しません。
  • 一方で、サイト配信やアクセス解析に伴う技術情報はインフラ提供者に処理される場合があります。

送らないもの

  • 入力テキスト本文
  • 差分結果そのもの
  • 本文処理のための外部 API リクエスト

保存方針

  • 入力テキストを localStorage や sessionStorage に保存しない
  • 入力内容を保持するための Cookie を使わない
  • アプリ自身は入力内容を自動保存しない

アクセス解析

このサイトでは Cloudflare Web Analytics を利用して、ページの閲覧数や表示速度など、サイト運用に必要な最小限の集計情報を把握します。

  • ページの閲覧数や表示速度などの集計情報
  • 閲覧ページ、参照元、デバイス種別、ブラウザなどの集計情報
  • 入力テキストや差分結果は、比較処理のためにサーバーへ送信しない

限界

この方針は、利用者の端末上のソフトウェア、ブラウザ拡張機能、ネットワーク環境による情報取得までは防ぎません。
